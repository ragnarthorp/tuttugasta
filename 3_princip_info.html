// SKJÖL 1914: UPPHAF HEIMSSTYRJALDAR //

Greining staðfest: Morðið á Franz Ferdinand erkihertoga í Sarajevó þann 28. júní 1914, framið af Gavrilo Princip, leysti úr læðingi keðjuverkun yfirlýsinga um stríð milli stórvelda Evrópu vegna flókinna hernaðarbandalaga.

Þetta markar upphaf Fyrri heimsstyrjaldarinnar (WWI), átaka sem áttu eftir að draga milljónir manna til dauða og umbreyta heiminum.

// NÝ GÖGN ÞARF: FLANDERS FIELDS //

Stríðið á vesturvígstöðvunum einkenndist af grimmilegum skotgrafahernaði. Milljónir féllu á stöðum eins og á engjum Flanders í Belgíu.

Á gröfum fallinna og sprengjugígum vígvallanna óx oft rauð blómategund sem varð síðar alþjóðlegt tákn til minningar um þá sem féllu í stríðinu, sérstaklega eftir frægt ljóð eftir John McCrae.
